## TLA+ (Temporal Logic of Actions) 拡張の機能

- TLA+:Parse Module
    - tla ファイルの構文解析と PlusCal コードの変換
- TLA+:Check Model with TLC
    - TLC モデルチェッカーを使用して TLA+ モデルを検証
- TLA+:Evaluate Expression
    - TLA+ 式の評価
    - Learn TLA+ で言及される Scratch.tla の式評価を行う際には、この機能を使用する
